# Протокол обслуживания экспертного агента

Этот документ описывает, как поддерживать знания вашего ИИ-ассистента в актуальном состоянии.

## Как запустить проверку
Чтобы «запустить» проверку актуальности, просто напишите мне одну из следующих команд:
- *"Проверь актуальность данных"*
- *"Обнови базу знаний"*
- *"Сверь инфо по M-Safer с сайтом"*
