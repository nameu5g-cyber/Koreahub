// turbo-all

Этот воркфлоу является сокращенной версией системы верификации.

### Действие:
1. Прочитать последние данные из `lib/agent-config.ts`.
2. Выполнить `search_web` по запросу: `South Korea telecom M-Safer 2026 news`.
3. Сравнить данные и вывести краткий отчет (Diff) прямо в чат.
4. Предложить обновление, если найдены расхождения.

### Пример использования:
Просто напишите `/verify` в чате.
